
## 快速开始

### 环境准备
* node14及以上版本
* 准备一个编辑器，推荐vscode
* 先读一下下面一篇文章（构想qwik）


## 使用CLI创建应用

终端执行下面命令，按选项来创建应用即可。
```javascript
npm create qwik@latest
```


## 在开发环境运行应用

应用创建好了之后，执行下面几步：

* cd your-project-name
* npm install
* npm start

你会在终端看到下面这几句提示：
```javascript

  VITE v3.0.2  ready in 140 ms

  ➜  Local:   http://localhost:5174/
  ➜  Network: use --host to expose
```

点击链接，用浏览器打开应用。



## 一些命令

这里有一些常用命令，完整列表可以去看package.json。
* npm start: 是npm run dev的别名
* npm run build: qwik build
